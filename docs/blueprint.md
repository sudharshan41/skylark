# **App Name**: Skylark AI Coordinator

## Core Features:

- Pilot Roster Query: Query pilot availability by skill, certification, and location.
- Pilot Status Update: Update pilot status (Available / On Leave / Unavailable) and sync to Google Sheets.
- Project Assignment Matching: Match pilots and drones to projects based on requirements.
- Drone Fleet Query: Query drone fleet by capability, availability, and location. Sync status to Google Sheets.
- Conflict Detection: Detect double-bookings, skill/certification mismatches, and equipment-pilot location mismatches using reasoning with LLM.
- Urgent Reassignment Tool: Propose optimal pilot/drone reassignments using generative AI to address urgent situations.
- Real-time Assignment Tracking: Monitor and track active assignments in real-time, providing status updates and alerts.

## Style Guidelines:

- Primary color: Deep blue (#2E3192) to evoke trust, precision, and a sense of the technological.
- Background color: Very light desaturated blue (#E8E9F3), visually of the same hue as the primary, for a clean backdrop.
- Accent color: Purple (#9C27B0) for highlights and calls to action.
- Font: 'Inter' for both headlines and body text; sans-serif, for a modern, machined, objective, neutral look.
- Use clean, modern icons to represent pilot skills, drone capabilities, and project statuses.
- Design a clear and intuitive dashboard for managing pilots, drones, and assignments, focusing on real-time updates and conflict alerts.
- Implement subtle animations to highlight important changes, such as new assignments or conflict detections.
